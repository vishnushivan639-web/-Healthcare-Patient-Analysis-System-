Healthcare Analytics Project
Healthcare Patient Analysis System
Overview
This project focuses on analyzing patient health data to extract meaningful insights and support better healthcare decision-making using data analytics techniques.
The system is designed using a structured and modular workflow that includes:
•	Data preprocessing and cleaning 
•	Exploratory Data Analysis (EDA) 
•	Data visualization 
•	Insight generation 
The main objective is to understand patient health patterns and identify key factors affecting overall health conditions.
________________________________________
Dataset
Source: Synthetic healthcare dataset (generated using Python libraries)
Description:
The dataset includes various patient health attributes such as:
•	Age 
•	Gender 
•	Blood Pressure 
•	Sugar Level 
•	Cholesterol 
•	Heart Rate 
It simulates real-world healthcare records for analysis purposes.
